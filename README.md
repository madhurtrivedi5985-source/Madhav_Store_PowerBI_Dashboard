# Madhav_Store_PowerBI_Dashboard
📊 Interactive Power BI dashboard for Madhav Ecommerce Sales Analysis using DAX, Power Query, and Data Modeling.

## Dashboard Preview

![Dashboard Preview](dashboard.png)

## 📌 Overview
An interactive **Power BI Dashboard** built to analyze Madhav Ecommerce sales performance. It provides insights into sales, profit, customer behavior, payment methods, and product performance through interactive visualizations.

## 🔑 Key Metrics
- 💰 **Total Sales Amount:** ₹438K
- 📈 **Total Profit:** ₹37K
- 📦 **Total Quantity Sold:** 5,615
- 💵 **Average Order Value (AOV):** ₹120.9K

## 📊 Dashboard Features
- **State-wise Sales Analysis**
  - Maharashtra
  - Madhya Pradesh
  - Uttar Pradesh
  - Gujarat

- **Category-wise Quantity Analysis**
  - Clothing
  - Electronics
  - Furniture

- **Monthly Profit Trend**
  - Identify profitable and loss-making months.

- **Customer Analysis**
  - Top customers based on sales amount.

- **Payment Mode Distribution**
  - Cash on Delivery (COD)
  - UPI
  - Credit Card
  - Debit Card
  - EMI

- **Sub-Category Profit Analysis**
  - Best-performing product sub-categories.

## 🛠️ Tech Stack
- Power BI Desktop
- Power Query
- DAX (Data Analysis Expressions)
- Data Modeling

## 🚀 Key Insights
- Maharashtra generated the highest sales.
- Clothing contributed the largest share of quantity sold.
- COD was the most preferred payment method.
- Printers generated the highest profit among sub-categories.
- Monthly profit analysis highlights seasonal performance trends.
